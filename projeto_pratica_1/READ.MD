Este é um projeto de análise de dados dados públicos fornecidos pelo governo do Estado de São Paulo. 





Os dados utilizados neste projeto foram extraídos do Portal de Dados Abertos do Estado de São Paulo [https://dadosabertos.sp.gov.br/dataset/frota-ativa/resource/6dca0323-ef57-45c9-921b-f75ad50c8bd1] e são de livre uso conforme o Decreto Estadual nº 58.052/2012.
